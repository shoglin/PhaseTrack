# Фазовые траектории

Исследование фазовых траекторий нелинейных динамических систем.

## Структура проекта

- **Point1** — начальный этап исследования
- **Point2** — развитие методов анализа
- **Point3** — финальная версия с полной визуализацией
- **Functions** — вспомогательные функции (поиск равновесий, проверка границ, 3D-визуализация)

## Исследуемая система

Реализована система дифференциальных уравнений Рабиновича–Фабриканта:

dx/dt = y(z - 1 + x²) + γx
dy/dt = x(3z + 1 - x²) + γy
dz/dt = -2z(α + xy)
